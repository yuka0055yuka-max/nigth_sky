# 幻想夜空 🌌

癒しと幻想をテーマにしたインタラクティブなCanvasアートです。  
夜空に静かに瞬く星々。  
画面をタップすると、星の配置が変化します。

---

## 使い方

1. Webブラウザでアクセス  
2. タップして夜空を感じる  

---

## 📦 PWA対応

- ホーム画面に追加可能（iOS / Android）  
- オフラインでも動作  
- Web共有で誰でもアクセス可能
